# Installation

First you're going to need to clone this repository to `~/.vim`

    git clone https://github.com/darthdeus/dotvim ~/.vim

Symlink the `vimrc`

    ln -nsf ~/.vim/vimrc ~/.vimrc

Install the bundle

    vim +:BundleInstall

**And enjoy the uber experience these dotfiles provide.**
